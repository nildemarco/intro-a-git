Comandos útiles de git

  • git init: comando para iniciar un repositorio de git local
  • git status: indica el estado de los archivos dentro del repositorio
  • git add: comando para agregar archivos al espacio de trabajo
    • Tengo que especificar o el nombre del archivo que quiero agregar, o escribir un punto "." para indicar que quiero guardar todos los archivos.
  • git diff: muestra las diferencias entre los archivos desde la última vez que hice un agregado de cambios con git add y los cambios actuales
  • :q : comando que me permite salir de la muestra de diferencias despues de usar git diff
  • git commit -m "Algun mensaje": me permite guardar los cambios en un commit e identificarlos con un titulo o mensaje
  • git push: me permite enviar los commits a un repositorio remoto en github
    • Cuando se usa por primera vez, debo escribir git push -u origin master
